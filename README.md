## ✅ Completed Tasks

### 🔢 Task 1 - Number Guessing Game
- A console-based game that generates a random number between 1 and 100.
- The user is prompted to guess the number.
- Feedback is given whether the guess is too high or too low.
- The game supports multiple rounds, a scoring system, and attempt limits.

### 🎓 Task 2 - Student Grade Calculator
- Takes marks for different subjects as input.
- Calculates total marks and average percentage.
- Assigns grades based on performance.
- Displays results including grade and percentage.

### 🏧 Task 3 - ATM Interface
- Simulates an ATM with basic functionalities.
- Includes features like deposit, withdraw, and balance check.
- Ensures input validation and proper account linking.
